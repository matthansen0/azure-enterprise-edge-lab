Additional WAF exercises (curl / terminal only)

These rules rely on HTTP headers that browsers cannot override in fetch(). Run them from a terminal.

# Header-based block (custom rule: BlockDemoHeader)
curl -s -o /dev/null -w "HTTP %{http_code}" -H "X-Demo-Block: true" \
  "https://<AFD_FQDN>/api/health"
# Expected: 403

# Bot User-Agent block (custom rule: BlockDemoBotUA)
curl -s -o /dev/null -w "HTTP %{http_code}" -A "DemoMaliciousBot/1.0" \
  "https://<AFD_FQDN>/api/health"
# Expected: 403
